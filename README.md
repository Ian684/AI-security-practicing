# AI-security-practicing

A personal practice repository for AI security and prompt-injection style challenges. The current writeups focus on Lakera Gandalf challenges and record prompts, observations, and solving strategies.

## Repository Structure

```text
AI-security-practicing/
├── gandalf_lakera_ai_password_reveal_writeup
├── gandalf_lakera_ai_reverse_gandalf_writeup
├── gandalf_lakera_ai_truthteller_gandalf_writeup
└── README.md
```

## Included Writeups

| File | Description |
|---|---|
| `gandalf_lakera_ai_password_reveal_writeup` | Notes for password-reveal style Gandalf levels. |
| `gandalf_lakera_ai_reverse_gandalf_writeup` | Notes for reverse-Gandalf style prompt challenge. |
| `gandalf_lakera_ai_truthteller_gandalf_writeup` | Notes for truth-teller Gandalf prompt challenge. |

## Main Topics

- Prompt injection
- Jailbreak-style reasoning
- Prompt constraint bypass
- Information extraction from LLM responses
- Defensive prompt behavior observation

## Usage

Clone the repository:

```bash
git clone https://github.com/Ian684/AI-security-practicing.git
cd AI-security-practicing
```

Open any writeup file to review the prompts and solving notes:

```bash
cat gandalf_lakera_ai_password_reveal_writeup
```

## Notes

- Some prompts may only work under specific challenge versions or model behaviors.
